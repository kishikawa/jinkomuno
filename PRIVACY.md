# プライバシーポリシー

本プライバシーポリシー（以下「本ポリシー」）は、Discord ボット「ムノ」（人工無能。以下「本ボット」）における利用者の情報の取り扱いについて定めるものです。本ボットを利用することで、利用者は本ポリシーに同意したものとみなされます。

> **注記:** 本ドキュメントは個人の趣味プロジェクト向けのサンプルです。実際の公開・運用にあたっては、運営形態に合わせて連絡先や事業者情報を加筆し、必要に応じて専門家の確認を受けることを推奨します。

最終更新日: 2026年6月2日

## 1. 収集する情報

本ボットは、本ボットが参加している Discord のサーバーおよびチャンネルにおいて送信されたメッセージの**テキスト本文**を取得します。これは、会話を学習し応答を生成するという本ボットの機能上必要なためです。

本ボットが**取得しない・保存しないもの**は次のとおりです。

- パスワード、認証トークン、メールアドレスその他の認証・連絡先情報
- クレジットカード情報等の決済情報
- メッセージ送信者の Discord ユーザー名・ユーザー ID（応答処理の過程で参照しますが、学習データとしては保存しません）
- 画像・音声・添付ファイルの内容

## 2. 利用目的

取得したメッセージのテキストは、以下の目的にのみ利用します。

- 本ボットの会話モデル（マルコフ連鎖）の学習
- 利用者の発言に対する応答の生成

## 3. 保存方法と保存期間

- 取得したテキストは形態素解析され、**単語のつながりの統計情報（N-gram）**として、運営者の管理する環境のローカルデータベース（SQLite）に保存されます。
- メッセージは原文そのままの形では保存されませんが、統計情報から文の断片が再構成されて応答に現れる可能性があります。この点をあらかじめご了承ください。
- 保存された学習データは、運営者が削除するまで保持されます。

## 4. 第三者への提供

本ボットは、取得した情報を販売・貸与しません。ただし、以下の場合に限り、メッセージのテキスト本文を外部サービスに送信することがあります。

- **LLM フォールバック機能が有効な場合:** マルコフ連鎖で適切な応答を生成できないとき、応答を生成する目的でメッセージのテキストを Anthropic, PBC の API に送信することがあります。送信されたデータの取り扱いは同社のポリシー（https://www.anthropic.com/legal/privacy ）に従います。
- この機能は運営者の設定により無効化でき、無効の場合は外部送信は一切行われません。
- 法令に基づく開示請求があった場合。

## 5. データの削除・お問い合わせ

学習データの削除をご希望の場合は、本リポジトリの GitHub Issue（https://github.com/kishikawa/jinkomuno/issues ）までご連絡ください。

ただし、学習データは個々の発言ではなく統計情報として混在して保存されるため、**特定の個人や特定のメッセージのみを選択的に削除することは技術的に困難**です。その場合は学習データベース全体の初期化という形での対応となることがあります。

## 6. セキュリティ

運営者は、取得した情報の漏えい・滅失・毀損の防止に努めます。なお、本ボットは認証情報等の秘密情報を取得・保存しない設計としています。

## 7. 未成年者の利用

未成年者が本ボットを利用する場合は、保護者の同意のもとで利用してください。

## 8. 本ポリシーの変更

運営者は、必要に応じて本ポリシーを変更することがあります。変更後の内容は本リポジトリ上で公開された時点で効力を生じます。

## 9. お問い合わせ

本ポリシーに関するお問い合わせは、本リポジトリの GitHub Issue（https://github.com/kishikawa/jinkomuno/issues ）までお願いします。
