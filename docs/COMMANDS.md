# コマンド一覧

Discord 上で人工無能「ムノ」を操作するためのコマンド。
すべて接頭辞 **`!muno`** で始める（半角スペース区切り）。

コマンド以外の通常の発言には、メンション・名前呼び・一定確率で自動応答し、
発言内容は学習に使われる。コマンドは学習されない。

## 一般コマンド（誰でも使える）

| コマンド | 説明 | 応答の例 |
|---|---|---|
| `!muno help` | コマンド一覧を表示する | コマンド一覧 |
| `!muno ping` | 生存確認 | 「生きてるけど。で、何？」 |
| `!muno stats` | 学習状況（語彙数・LLM の有無・このチャンネルの状態）を表示 | 「語彙 169 / LLM OFF / ここでは「通常」」 |
| `!muno say` | お題なしでひとこと生成させる | 「で、結局なにがしたいの」 |
| `!muno teach <文>` | 指定した文を学習させる | 「覚えた。記憶力には期待しないでよ」 |

### 例

```
!muno teach 今日は最高にだるい
!muno say
!muno stats
```

## 管理コマンド（サーバーの管理権限が必要）

権限は「管理者（Administrator）」または「サーバー管理（Manage Server）」で判定する。
権限がない利用者が実行すると拒否される。

| コマンド | 説明 | 注意 |
|---|---|---|
| `!muno mute` | このチャンネルで**自発的な発言を止める** | メンション・名前呼びには引き続き反応する |
| `!muno unmute` | ミュートを解除する | — |
| `!muno forget` | **学習データを全消去**する | 元に戻せない。実行前に確認すること |

### ミュートについて

- ミュートは**チャンネル単位**で、メモリ上にのみ保持される。
- ボットを**再起動すると解除**される（永続化していない）。

## 補足

- 接頭辞は `brain/commands.py` の `PREFIX` で変更できる。
- コマンドの応答文や追加コマンドも同ファイルで編集・拡張できる。
- データの取り扱いは[プライバシーポリシー](../PRIVACY.md)、利用条件は[利用規約](../TERMS.md)を参照。
